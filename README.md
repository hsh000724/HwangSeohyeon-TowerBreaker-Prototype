# 🎮 TowerBreaker Core Loop Prototype

Unity 기반 액션 RPG 레퍼런스 게임의 코어 루프(전투 → 보상 → 성장 → 전투)를 구현한 프로토타입입니다.

---

## 🛠 제작 기간
2026-03-13 ~ 2026-03-15 (3일)

---

## 🛠 개발 환경
- Unity (버전)
- C#
- 플랫폼: PC

---

## 🎯 구현 목표
- 전투 → 보상 → 성장 → 전투 구조 구현
- 타격감, 조작감 중심의 전투 시스템 설계

---

## ⚔ 주요 기능

### 1. 플레이어
- 대시 (우클릭)
- 공격 (좌클릭, 연속 입력)
- 방어 (Space, 넉백)

### 2. 전투 시스템
- 물리 기반 넉백
- 벽 충돌 시 피격 처리
- 무적 상태 → 입력으로 해제

### 3. 적 AI
- 좌측 이동
- 넉백 상태 처리
- 오브젝트 풀링 기반 스폰

---

## 🔧 기술적 구현 포인트

### ✔ 넉백 시스템
- AddForce 기반 물리 넉백
- 다수 적에도 동일한 힘 적용

### ✔ 피격 판정
- FixedUpdate 기반 충돌 감지
- 벽 + 적 동시 충돌 시 데미지 처리

### ✔ 상태 관리
- isInvincible / isPlayerHit 분리
- 입력 기반 상태 해제 구조

---

### ✔ 보상 시스템 (상자 드랍 & 오픈)
- 적 처치 시 확률 기반 상자 드랍
- 상자 획득 시 일괄 개봉 처리
- 개봉 결과를 UI 텍스트로 즉시 피드백

### ✔ 장비 시스템
- 장비 등급 및 능력치 데이터 구조 설계
- 아웃게임에서 장비 목록 확인 및 장착 기능 구현
- 장비 장착 시 플레이어 스탯에 실시간 반영

### ✔ 자동 업그레이드 시스템
- 동일 장비 일정 개수 획득 시 상위 등급으로 자동 승급
- 단계별 능력치 증가 로직 설계 (지수 증가 구조)

---

## 🚀 개선 경험

- 넉백 힘 분산 문제 해결
- 피격 판정 누락 문제 수정
- 프리즈 → 무적 시스템으로 구조 개선
